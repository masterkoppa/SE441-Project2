README


Instructions
============

To execute the program run the provided batch script CompileAndRun.sh
this should set the class path and execute everything correctly.

Config
======
Inside the Drive.scala file there are 3 configurable settings:

numQueues 	- The number of queues in the system 	Default: 3

passengersPerDay	- The number of passengers that will go 
		  through the system per day 		Default: 2
numDays		- The number of days you would like to
		  simulate				Default: 2
